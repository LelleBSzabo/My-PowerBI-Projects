# 📊 Power BI Retail & Sales Data Analysis Project

This project was completed as part of a **Data Technician Bootcamp** and focuses on analysing and visualising datasets using **Microsoft Power BI**. The objective was to transform raw data into meaningful insights through **data cleaning, transformation, interactive reports, and visual storytelling**.

Using Power BI, this project explores **retail and sales data** to uncover trends, compare performance across categories and regions, and present insights through interactive dashboards.

---

## 🚀 Project Overview

In this project, Power BI was used to:

- Import and combine datasets from multiple sources
- Clean and transform data using **Power Query**
- Create **interactive reports and dashboards**
- Build **DAX calculated columns and measures**
- Visualise key trends using a variety of charts

The final reports allow users to explore data dynamically using **filters, slicers, and interactive visualisations**.

---

## 🛠️ Power BI Skills Demonstrated

### Data Transformation with Power Query

Power Query was used to prepare datasets for analysis by:

- Cleaning and structuring raw data
- Renaming columns and correcting data types
- Removing duplicates and unnecessary fields
- Transforming and combining data from multiple sources

These transformations ensured the data was **accurate, structured, and ready for analysis**.

---

### DAX (Data Analysis Expressions)

DAX was used to create calculated columns and measures that support deeper analysis.

Examples include:

- Aggregated sales calculations
- Conditional logic for categorising data
- Percentage calculations and KPIs

These calculations enable **more advanced analysis and dynamic reporting** within Power BI.

---

## 📊 Visualisations Created

Multiple visualisation types were used to present the data clearly and effectively:

- **Bar Charts** – Compare sales performance across categories
- **Line Charts** – Display trends over time
- **Pie Charts** – Show proportional distributions
- **Maps** – Visualise geographic sales performance
- **Tables and KPI visuals** – Provide detailed metrics

These visuals were combined into interactive reports to create a **comprehensive overview of sales and retail data**.

---

## 🎛️ Interactive Report Features

The reports include interactive elements such as:

- **Slicers** for filtering by category, region, or other attributes
- **Filters** for narrowing down datasets
- Cross-filtering between visualisations
- Dynamic updates when selections are changed

These features allow users to **explore the data from multiple perspectives**.

The dashboards highlight:

This approach helps transform raw data into **clear, actionable insights**.
